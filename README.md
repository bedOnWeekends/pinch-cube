# Description
This is a simple program to operate 3x3x3 cube with pinch gesture.
when you pinch both hand and zoom it, the cube will explode.
in opposite way, it will be shrink. 

# Environment (verified)
- Python: 3.10.11
- opencv-python: 4.13.0
- mediapipe: 0.10.14
- pygame: 2.6.1
- PyOpenGL: 3.1.10

# How To Play?

## 1. download libraries
```
pip install opencv-python==4.13.0 mediapipe==0.10.14 pygame==2.6.1 PyOpenGL==3.1.10 PyOpenGL_accelerate==3.1.10
```
## 2. play it
```
python HandTrackingCube.py
```
